This directory contains example code demonstrating how to use the DCGAN project.

## Available Examples

### `example_usage.py`
Comprehensive examples showing:
- Basic training setup
- Image generation from trained models
- Using custom datasets

Run the example:
```bash
python examples/example_usage.py
```

## Learning Resources

These examples are designed to help you understand:
1. How to initialize and configure DCGAN models
2. How to train models on your own datasets
3. How to generate images from trained models
4. Best practices for working with the codebase

## Customization

Feel free to modify these examples to suit your needs. They serve as starting points for your own projects.
