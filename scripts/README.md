This directory contains utility scripts for the DCGAN project.

## Available Scripts

### `visualize_training.py`
Visualize training progress from saved checkpoints.

```bash
python scripts/visualize_training.py --checkpoint_dir checkpoints/
```

### `compare_models.py`
Compare images generated by different trained models.

```bash
python scripts/compare_models.py --checkpoints checkpoints/gen1.pth checkpoints/gen2.pth
```

## Usage

All scripts can be run from the project root directory. Make sure to install all dependencies first:

```bash
pip install -r requirements.txt
```
