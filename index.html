About This Project

This project implements DCGAN (Deep Convolutional GAN) for generating realistic images. The architecture uses convolutional layers in both generator and discriminator networks. It includes batch normalization, LeakyReLU activations, and proper weight initialization for stable training. Perfect for learning GAN fundamentals and image generation.

Features

DCGAN Architecture

Convolutional generator and discriminator networks with proper architecture design.

Adversarial Training

Stable adversarial training techniques for realistic image generation.

Batch Normalization

Batch normalization and LeakyReLU activations for stable training.

Realistic Images

Generate high-quality, realistic images from random noise vectors.

Quick Start

Installation

pip install -r requirements.txt

Training

Run the Jupyter notebook or use the Python script:

# Using Jupyter Notebook jupyter notebook dcgan_training.ipynb # Or using Python script python main.py
